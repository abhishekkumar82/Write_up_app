## Run Locally


### Clone this Project

```
https://github.com/Swati-Tanu/Write-Up.git
```

### Install npm Packages

```javascript
npm i --global
```

### Run Server
```javascript
npx nodemon app.js
```
### Runs the project in the development mode

[http://localhost:9090](http://localhost:9090)


### Environment Variables Required

`PORT`

`MONGODB_URI`

`GOOGLE_CLIENT_ID`

`GOOGLE_CLIENT_SECRET`

`GOOGLE_CALLBACK_URL`
